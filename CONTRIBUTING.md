# 🤝 Contributing

Create a focused branch, add behavior-driven tests, run the complete local checks, and open a pull request describing operational impact, security implications, accessibility impact, and rollback. Use conventional commits. Keep domain rules out of UI components and route handlers. New endpoints require schemas, authorization, OpenAPI detail, tests, and audit consideration. New visual interactions require keyboard, screen reader, high contrast, reduced motion, mobile, and wide-display verification. Never commit secrets, real spectator data, or venue-sensitive security plans.
